Quick Overview

LSM Tree

Write-Optimized: Converts random writes to sequential writes. Perfect for write-heavy workloads, logging systems, and time-series data.

Used in: Cassandra, RocksDB, LevelDB, HBase

B+ Tree

Read-Optimized: Balanced tree with excellent point queries and range scans. The foundation of relational databases.

Used in: MySQL, PostgreSQL, SQLite, SQL Server

HBase

Distributed System: Production-ready distributed database built on LSM Trees. Provides fault tolerance, high availability, and scalability.

Components: HMaster, RegionServers, ZooKeeper, HDFS
